### code-review

官方插件：代码审查

8.8k

- 功能：专门用于对代码库做全面审查，关注质量、安全、性能、可维护性等

- 适用场景：
  - 提 PR 前自查
  - 帮同事做 Code Review
  - 遗留代码梳理
- 典型用法：
  - 在 Claude Code 里：

```
    请使用 code-review 技能，审查当前项目 src/ 目录下的代码，重点指出：
    - 潜在安全漏洞
    - 性能问题
    - 可维护性差的模式
    并按优先级排序。
```

https://github.com/anthropics/claude-plugins-official/tree/main/plugins/code-review
